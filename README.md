
# MS-DOS-Temperature-Converter
An MS-DOS version of my Python temperature converter program that I created using the C programming language, as a fun project to see what it was like being an old school PC programmer during the late 80s and early 90s when MS-DOS was relevant.

No AI-generated code was used during the making of this program.


# How to run the program
You will need to have the DOSBOX emulator installed. I highly recommend DOSBOX-X due to the convenient features it has over the origianl version of the emulator. If you're not familiar with DOSBOX or using MS-DOS commands, the following steps below will help you install the emulator and run the program right away.


1. Go to https://github.com/joncampbell123/dosbox-x/releases/tag/dosbox-x-v2026.07.02

2. Choose "dosbox-x-windows-2026.07.02-setup.exe"(Unless you're using an operating system other than Windows)

3. If you see a pop up that says "Windows protected your PC", click "More info" and click "Run anyway". Don't worry, this file won't harm your computer.

4. Once the setup window appears, leave the default options the way they are and keep clicking "Next".

5. Click Install and wait for until the installation process is complete.

6. Click "Next", uncheck "View README.txt" and "Launch DOSBox-X", then click "Finish"

7. Go to where you placed your TEMPCONV.EXE file. By default it should be in the Downloads folder on your PC.

8. Right click the exe file and choose "Run with DosBox-X"(NOTE: If you're on Windows 11, make sure to click "Show more options")

9. You can now try my program without having to go through the hassle of learning the various MS-DOS commands.

# How to compile the program
You will need Borland Turbo C 2.01 to modify and compile the program. You can download the program from here and run it in DOSBOX-X: https://archive.org/details/msdos_borland_turbo_c_2.01


The version of the C language used is Borland's own dialect created exclusively for Turbo C 2.01. The links below well help you learn how to program in the Turbo C IDE, if you're interested.
